# 🛡️ Trademark Guardian

---

# Description

**Trademark Guardian** is a modern React web application built with TanStack Start, TanStack Router, and TanStack Query. It utilizes the comprehensive `shadcn/ui` component library alongside Tailwind CSS v4 for enterprise-grade styling and data management. 

---

# 🚀 Features

- **TanStack Start Framework:** Server-side rendering capable React framework using TanStack Router.
- **Full shadcn/ui Component Suite:** Comprehensive set of Radix UI primitives.
- **Data Visualization:** `recharts` integration for beautiful, responsive chart components.
- **Form Management:** `react-hook-form` with Zod schema validation.
- **Date Handling:** `date-fns` and `react-day-picker`.

---

# 🛠️ Tech Stack

| Layer | Technology |
| --- | --- |
| **Framework** | TanStack Start, TanStack Router |
| **UI Components** | shadcn/ui, Radix UI |
| **Styling** | Tailwind CSS v4 |
| **Data Fetching** | TanStack Query |
| **Runtime** | Bun |

---

# 🏎️ Quick Start

```bash
bun install
bun dev
```
